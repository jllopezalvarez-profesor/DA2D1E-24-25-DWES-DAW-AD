Subida de ficheros simple, sin DTO, con un solo fichero

En este formulario hay dos campos de texto, y uno de fichero. En el controlador los campos se procesan con @RequestParam.

El campo para el fichero se procesa de forma independiente. Si hay varios campos de fichero, cada campo será un parámetro independiente en el controlador, siempre que tengan distinto nombre (atributo "name"). Para el caso en el que los campos de fichero tienen el mismo nombre, ver el ejemplo simple con varios campos de fichero.
