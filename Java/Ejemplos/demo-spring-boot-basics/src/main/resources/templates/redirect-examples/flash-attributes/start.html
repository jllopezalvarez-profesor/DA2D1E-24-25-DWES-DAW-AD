Ejemplo de uso de flash attributes en redirecciones

Esta página se ha generado:

Desde esta página, se navega con un enlace (GET) a una página con dos formularios. En esa página, cada envío (POST) del formulario, fija un mensaje y redirige de nuevo de vuelta a este punto inicial

Estos mensajes se fijan en los POST usando flash attributes, y se recuperan en la plantilla de esta página para mostrarlos como un TOAST de Bootstrap.

Solo aparecerán los mensajes si se han fijado. Si no, como es el caso de acceder directamente a esta página, no aparecerán.
