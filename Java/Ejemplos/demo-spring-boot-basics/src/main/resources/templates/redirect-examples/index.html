Prueba de redirecciones

Desde esta página se puede acceder a dos páginas distintas:

  • Página que simula el detalle de un producto. Recibe el id del producto como @PathVariable: Detalle del producto con id 100
  • Página que simula el listado de productos de una categoría. Recibe el id de categoría como @PathVariable: Listado de la categoría con id 'INF'

En estas páginas hay enlaces para llamar a un método de controlador para añadir el producto a carro de compra, y volver a la misma página.

Habrá enlaces de distinto tipo, para ver distintas formas de redirigir
